@climateinteractive/en-roads-core / ModelContext

Class: ModelContext #

Defines a context that holds a distinct set of model inputs and outputs. These inputs and outputs are kept separate from those in other contexts, which allows an application to use the same underlying model to run with multiple I/O contexts.
