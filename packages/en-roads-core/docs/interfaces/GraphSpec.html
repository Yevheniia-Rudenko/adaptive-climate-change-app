@climateinteractive/en-roads-core / GraphSpec

Interface: GraphSpec #

Describes a graph that plots one or more model output variables.

Properties #

id #

Readonly id: GraphId

The graph ID.

kind #

Readonly kind: GraphKind

The graph kind.

titleKey #

Readonly titleKey: StringKey

The key for the graph title string.

miniTitleKey #

Optional Readonly miniTitleKey: StringKey

The key for the graph title as it appears in the miniature graph view (if undefined, use titleKey).

menuTitleKey #

Optional Readonly menuTitleKey: StringKey

The key for the graph title as it appears in the menu (if undefined, use titleKey).

subtitleKey #

Optional Readonly subtitleKey: StringKey

The key for the optional graph subtitle string.

groupingKey #

Optional Readonly groupingKey: StringKey

The key for the parent grouping in the menu.

descriptionKey #

Optional Readonly descriptionKey: StringKey

The key for the graph description string.

simulationModes #

Optional Readonly simulationModes: readonly SimulationModeId[]

The set of simulation modes in which this graph appears.

side #

Optional Readonly side: GraphSide

Whether the graph is shown on the left or right side of the main interface.

unitSystem #

Optional Readonly unitSystem: UnitSystem

The unit system for this graph (if undefined, assume metric or international units).

alternates #

Optional Readonly alternates: readonly GraphAlternateSpec[]

Alternate versions of this graph (e.g. in a different unit system).

xMin #

Optional Readonly xMin: number

The minimum x-axis value.

xMax #

Optional Readonly xMax: number

The maximum x-axis value.

xAxisLabelKey #

Optional Readonly xAxisLabelKey: StringKey

The key for the x-axis label string.

yMin #

Optional Readonly yMin: number

The minimum y-axis value.

yMax #

Optional Readonly yMax: number

The maximum y-axis value.

ySoftMin #

Optional Readonly ySoftMin: number

The “soft” minimum y-axis value. If defined, the y-axis minimum will not rise above this value, but will decrease as needed if any y values are below this value.

ySoftMax #

Optional Readonly ySoftMax: number

The “soft” maximum y-axis value. If defined, the y-axis maximum will not fall below this value, but will increase as needed if any y values are above this value.

yAxisLabelKey #

Optional Readonly yAxisLabelKey: StringKey

The key for the y-axis label string.

yFormat #

Optional Readonly yFormat: FormatString

The string used to format y-axis values.

datasets #

Readonly datasets: readonly GraphDatasetSpec[]

The datasets to plot in this graph.

legendItems #

Readonly legendItems: readonly GraphLegendItemSpec[]

The items to display in the legend for this graph.

dataTable #

Optional Readonly dataTable: DataTableSpec

The spec for the data table that should be shown in place of the graph.
