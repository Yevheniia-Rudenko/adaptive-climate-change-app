setAnimationsEnabled #

setAnimationsEnabled(enabled): void

Set whether animations should be enabled.

Note that if you want to disable animations entirely, it is better to set animations to false in GraphViewOptions to avoid initializing animation support in the underlying chart library. This method is mainly useful in cases where animations are disabled initially but you want to enable them later.

updateLabels #

updateLabels(): void

Update all labels to use the appropriate translated strings. This should be called after the current locale/language has changed to ensure that the labels are updated. This will also cause the chart to refresh its ticks and tooltips to use the correct locale-specific formatting based on the current locale value.

void
