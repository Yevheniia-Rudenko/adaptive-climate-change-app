@climateinteractive/sim-ui-graph / GraphStyle

Interface: GraphStyle #

Options for graph view styling.

Properties #

font #

Optional font: GraphFontStyle

The general font style for the graph.

xAxis #

Optional xAxis: GraphAxisStyle

The style for the x axis.

yAxis #

Optional yAxis: GraphAxisStyle

The style for the y axis.

Methods #

getAxisLabelFontSize #

Optional getAxisLabelFontSize(): number

Optional callback to customize the font size for axis labels. If defined, this will be called after layout events (e.g. after the browser window is resized.)

Returns #

number

The font size for axis labels, in pixels.

getTickLabelFontSize #

Optional getTickLabelFontSize(): number

Optional callback to customize the font size for tick labels. If defined, this will be called after layout events (e.g. after the browser window is resized.)

Returns #

number

The font size for tick labels, in pixels.

getDefaultLineWidth #

Optional getDefaultLineWidth(): number

Optional callback to customize the default line width for graph datasets. If defined, this will be called after layout events (e.g., after the browser window is resized). If undefined, a default value will be used.

Returns #

number

The default line width for graph datasets, in pixels.

getDatasetStyle #

Optional getDatasetStyle(datasetSpec, lineWidthInPx): GraphDatasetStyle

Optional callback to customize graph dataset style parameters. If defined, this will be called after layout events (e.g., after the browser window is resized).

Parameters #

Name Type Description
datasetSpec GraphDatasetSpec The spec for the dataset to be styled.
lineWidthInPx number The default line width. This will be the value provided by getDefaultLineWidth if defined, otherwise will be a default value.

Returns #

GraphDatasetStyle

The style for the given dataset.
